# Equinox_cooling_chamber
 Most pharmaceutical companies use a digital cooling chamber which is similar to a  refrigerator to keep the tablets and maintain the temperature,   Here comes our equinox cooling chamber which will ease the monitoring as it warns us  if the temperature is not in between minimum and maximum threshold temperature. 
to run this project first of all we need 
step1- make the hardware circuit (using bolt iot kit)
step2- connect the hardware circuit with the bolt cloud and make a product
step3- now open the linux server 
step4- make these three files
step5- now run the files.
